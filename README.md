# 🌦️ Skycast

Skycast is a modern weather application that provides real-time weather information using a weather API. The application features a dynamic user interface that automatically changes its theme and visuals based on the current weather conditions, creating an engaging and immersive user experience.

---

## ✨ Features

- 🌍 Real-time weather data
- 🔍 Search weather by city name
- 🎨 Dynamic themes based on weather conditions
- 🌡️ Displays temperature, humidity, wind speed, and weather description
- 📱 Fully responsive design
- ⚡ Fast and lightweight interface
- 🌈 Modern UI with smooth transitions

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Weather API

---

## 🚀 How It Works

1. Enter the name of any city.
2. The application fetches live weather data using a weather API.
3. Based on the returned weather condition (Sunny, Rainy, Cloudy, Snow, etc.), the website automatically updates its colors, backgrounds, and visual theme.
4. Weather details are displayed in an easy-to-read and responsive layout.
